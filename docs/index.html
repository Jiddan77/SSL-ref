About This Analysis

This system analyzes publicly available SSL match data to identify statistical patterns in referee decision-making. Our goal is to promote transparency and consistency in floorball officiating.

Methodology

  • Data collected from official SSL match statistics
  • Statistical significance testing for all bias claims
  • Focus on patterns, not individual performance
  • Continuous updates as new matches are played

Ethical Guidelines

This analysis is designed to support the floorball community by improving officiating consistency. It should not be used for personal attacks on referees or for gambling purposes.
